# Clear File Association

Windows Explorer shell extension that lets you clear/reset file type associations.

## Download

**[Download latest installer](https://github.com/mstsirkin/ClearFileAssoc/releases/latest)**

## Requirements

- Windows 10/11
- .NET Framework 4.8

## Usage

1. Run the installer as Administrator
2. Right-click any file
3. Click "Clear .xyz association"
4. The file type will now use system default or prompt you to choose an app
